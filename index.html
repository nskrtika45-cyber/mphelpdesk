1
Go to console.firebase.google.com → click Add project → name it MPHelpDesk → disable Google Analytics → Create project
2
In your project → Authentication → Get started → Email/Password → Enable → Save
3
Go to Firestore Database → Create database → Start in test mode → choose a region → Enable
4
Go to Project settings (gear icon) → Your apps → click </> Web → register app as MPHelpDesk → copy the firebaseConfig object values below
